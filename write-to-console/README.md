## Prerequisites

None

## Description

This program uses a single [**console.log**] function call to write "Hello World!" to the standard output.
Nothing fancy.

## Usage

```
$ npm run -s write-to-console
```

## Arguments and flags

None

## Example run

```
$ npm run -s write-to-console
Hello World!
```
